= KMeans

Attempting to build a fast, memory efficient K-Means program.

== How To Use

  data = [[1,1], [1,2], [1,1], [10000, 10000], [250, 250]]
  kmeans = KMeans.new(@data, :centroids => 4)
  kmeans.inspect
  => [[3, 4], [0, 1, 2]]
